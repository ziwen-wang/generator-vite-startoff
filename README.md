## Background
To help quickly generate project templates, just select h5&pc

## Install

Install Yeoman and generator-vite-startoff (we assume you have pre-installed node.js).

```bash
npm install -g yo
npm install -g generator-vite-startoff
```

## Usage

```bash
yo vite-startoff
```
